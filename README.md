# SpotifyWebApi - Core
A .net Core wrapper for the Spotify Web Api.

# Nuget package
Installing the package is easy.

Using the package manager console:
> `PM> Install-Package SpotifyWebApi-Core`


Using the dotnet CLI:
> `> dotnet add package SpotifyWebApi-Core`
